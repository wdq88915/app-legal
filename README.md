# app-legal

Static legal pages (Privacy Policy / Terms of Use) for iOS apps.

## Structure

```text
app-legal/
  index.html
  watermark-camera/
    privacy.html
    terms.html
```

## Customize

- Update app names if needed.
- Replace `support@yourdomain.com` with your real support email.
- Update the “Last updated” date when you make changes.

## Publish with GitHub Pages

1. Push this repository to GitHub.
2. In GitHub: **Settings → Pages**.
3. Set **Source** to **Deploy from a branch**.
4. Select branch **main** and folder **/(root)**.

After deployment, your pages will be available at:

- `https://<username>.github.io/<repo>/`
- `https://<username>.github.io/<repo>/watermark-camera/privacy.html`
- `https://<username>.github.io/<repo>/watermark-camera/terms.html`
